# harshRn.github.io

Legacy GitHub Pages host. Serves a single `index.html` that redirects
to **[jiqirens.net](https://jiqirens.net/)**, the active Jīqìréns
landing site (Astro + Cloudflare Pages).

Kept alive so older inbound links — including Glossa Traductor's App
Store Marketing URL — keep working. Will be retired once App Store
Connect metadata is updated to point at jiqirens.net directly.

The product privacy and support pages at
`https://harshrn.github.io/traductor/*` are served from a separate
repo and are not affected by anything here.
